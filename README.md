##  Proyecto Destacado / Capstone Project

### **ConnectSystem — SaaS Multi-Tenant de Inventario y CRM**
*Plataforma comercial B2B para PYMEs que unifica stock en tiempo real, histórico de almacenes, pipeline de ventas y facturación automatizada. Desplegado en producción con arquitectura Cloud de coste mensual cero.*

*   **Arquitectura Avanzada:** Implementación de estructura Multi-Tenant con aislamiento de datos y control de acceso basado en roles (RBAC) mediante **Spring Security y tokens JWT**[cite: 1].
*   **Frontend de Vanguardia:** SPA/SSR interactiva construida con **Next.js (App Router)**, **React 19** y TypeScript estricto[cite: 1].
*   **Fintech & Facturación:** Integración de pasarela de pagos mediante **Stripe SDK** (Checkout y Webhooks automatizados) y generación dinámica de auditorías de facturas en PDF con **OpenPDF**[cite: 1].
*   **Ingeniería de Rendimiento en Base de Datos (PostgreSQL):**[cite: 1]
    *   Eliminación del problema **N+1 queries** en relaciones complejas mediante estrategias de `FetchType.LAZY` y sentencias `JOIN FETCH` explícitas[cite: 1].
    *   Optimización del pool de conexiones bajo alta concurrencia desactivando el comportamiento nativo de `spring.jpa.open-in-view`[cite: 1].
    *   Reducción drástica de latencias de lectura mediante indexación inteligente (`@Index`) en campos clave correlacionados (`tenant_id`, `user_id`)[cite: 1].
